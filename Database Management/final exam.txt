SELECT i.Name, SUM(c.Credits) AS TotalCredits
FROM Instructor i
JOIN Course c ON i.InstructorID = c.InstructorID
GROUP BY i.InstructorID, i.Name
ORDER BY TotalCredits DESC
LIMIT 1;

FROM Student S
JOIN Enrollment E ON S.StudentID = E.StudentID
GROUP BY S.StudentID
HAVING COUNT(E.EnrollmentID) > 2;

GROUP BY is used to group rows that have the same values in specified columns. Often used with aggregate functions.


Example: SUM(), AVG()
SELECT CustomerID, COUNT(*) AS TotalOrders
FROM Orders
GROUP BY CustomerID;


ORDER BY is used to sort the result set based on one or more columns.
Example:
SELECT * FROM Products
ORDER BY Price DESC;
